## 📚 본문


### Reflect 를 활용한 인스턴스 생성
```java
Class clazz = Class.forName("클래스풀네임");

// Java 9 이후 권장 방법
Object obj = clazz.getDeclaredConstructor().newInstance();
// 또는 매개변수가 있는 생성자의 경우
Object obj2 = clazz.getDeclaredConstructor(String.class, int.class).newInstance("값", 10);
```
